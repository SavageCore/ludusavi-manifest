# ludusavi-manifest

Additional [Ludusavi](https://github.com/mtkennerly/ludusavi) manifest for finding save location of [various](#supported-emulators) emulators on Linux.

## How To Use:

### Add manifest to Ludusavi

Select the `Other` tab and add the new manifest to the list.

URL to add: `https://raw.githubusercontent.com/SavageCore/ludusavi-manifest/main/manifest.yml`

### Update roots

Ensure you create a `Home folder` root with the path of `/home/username/`

## Supported Emulators:

- Dolphin (Flatpak)
- Eden (AppImage)
- Ryujinx (AppImage)
